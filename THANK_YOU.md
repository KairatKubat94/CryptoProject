
### 3. Добавить новые алгоритмы

- Откройте `crypto_algorithms.py`
- Добавьте новый класс (например `DESCipher`)

- Добавьте UI в `index.html`

- Измените цвета в `styles.css`

- Модифицируйте макет в `index.html`
