## 🚀 Как запустить

### Способ 1: Автоматический (Windows) — САМЫЙ БЫСТРЫЙ

```bash
run.bat
```

Все установится и запустится автоматически!

### Способ 2: Ручной запуск (все ОС)

```bash
# 1. Перейти в папку
cd CryptoPr

# 2. Создать окружение
python -m venv venv

# 3. Активировать окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить сервер
python app.py
```

### После запуска

1. Откройте браузер
2. Перейдите на **http://127.0.0.1:5000**
3. Выберите алгоритм (Caesar, AES или RSA)
4. Введите данные
5. Смотрите пошаговую визуализацию!

## 🧪 Что попробовать

### 📝 Caesar Cipher

- Введите текст: `Hello World`
- Смещение: `3`
- Результат: `Khoor Zruog`
- Видно: каждую букву шифруют отдельно

### 🔒 AES Cipher

- Введите текст: `Secret Message`
- Пароль: `mypassword`
- Результат: `gAAAAABqAiLNP9RUeVq_YxX7v...`
- Видно: 5 этапов шифрования

### 🔑 RSA Cipher

- Нажмите "Сгенерировать ключи" (ждите)
- Введите текст: `Secret`
- Результат: `ac8fb38af2177a2c0e5386cf9c...`
- Видно: 6 этапов генерирования ключей + 5 шагов шифрования

## 🤔 Часто задаваемые вопросы

**Q: Почему Caesar медленнее всех?**
A: Визуально показываем каждую букву. В реальности он самый быстрый.

**Q: Может ли RSA взломать?**
A: Теоретически да, практически нет. Перебор займет больше времени чем возраст Вселенной.

**Q: Нужна ли база данных?**
A: Нет, приложение не сохраняет данные. Всё в памяти.

**Q: Можно ли использовать для своего проекта?**
A: Да! Расширяйте и используйте как хотите.

## 🐛 Если что-то не работает

**Проблема:** `ModuleNotFoundError: No module named 'flask'`

```bash
# Решение:
pip install -r requirements.txt
```

**Проблема:** `Address already in use`

```bash
# Решение: используйте другой порт
# Изменить в app.py:
app.run(debug=True, host='127.0.0.1', port=8000)
```

**Проблема:** Сервер медленный

```bash
# Решение: нормально для dev режима
# В production используйте gunicorn:
pip install gunicorn
gunicorn app:app
```
