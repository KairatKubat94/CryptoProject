## 🚀 Действия

### Запустить приложение

```bash
# Вариант 1: Windows (самый быстрый)
run.bat

# Вариант 2: Ручной запуск
python -m venv venv
venv\Scripts\activate  # или: source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Затем откройте: **http://127.0.0.1:5000**

### Тестировать через API

```bash
# Отправить запрос на шифрование Caesar
curl -X POST http://127.0.0.1:5000/api/caesar/encrypt \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello", "shift": 3}'
```

Примеры для всех алгоритмов в [TESTS.md](TESTS.md)

### Добавить новый алгоритм

1. Откройте `crypto_algorithms.py`
2. Добавьте новый класс (например `DESCipher`)
3. Откройте `app.py` и добавьте endpoint
4. Откройте `index.html` и добавьте UI
5. Готово!

## 🤝 Помощь и поддержка

### Если что-то не работает

3. Смотрите терминал где запустили сервер

1. Добавьте новый алгоритм в `crypto_algorithms.py`
2. Добавьте endpoint в `app.py`
