# 🚀 Быстрый старт

## Установка и запуск (2 минуты)

### Вариант 1: Автоматический запуск (Windows)

Просто запустите **`run.bat`** — остальное сделается автоматически:

```bash
run.bat
```

### Вариант 2: Ручной запуск

```bash
# 1. Перейти в папку проекта
cd CryptoPr

# 2. Создать виртуальное окружение
python -m venv venv

# 3. Активировать окружение
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить сервер
python app.py
```

## 🌐 Открыть приложение

После запуска сервера откройте браузер и перейдите на:

```
http://127.0.0.1:5000
```

## 🧪 Примеры тестирования

### 📝 Caesar Cipher

- Введите текст: `Hello World`
- Смещение: `3`
- Нажмите "Шифровать"
- Результат: `Khoor Zruog`

### 🔒 AES Cipher

- Введите текст: `Secret Message`
- Пароль: `mypassword`
- Нажмите "Шифровать"
- Скопируйте результат
- Нажмите "Расшифровать"
- Введите тот же пароль
- Результат: `Secret Message`

### 🔑 RSA Cipher

- Нажмите "Сгенерировать ключи" (ждите 5-10 сек)
- Введите текст: `Secret`
- Нажмите "Шифровать" (ждите 5-10 сек)
- Видно зашифрованное значение

## 💡 Советы

- Caesar — для обучения, не для реального использования
- AES — используйте для защиты данных
- RSA — используйте для подписей и обмена ключами
- Комбинируйте RSA + AES для полной защиты

## 🐛 Решение проблем

**Проблема:** Сервер не запускается

```
Решение: Убедитесь, что Python установлен и добавлен в PATH
python --version
```

**Проблема:** Ошибка при установке зависимостей

```
Решение: Обновите pip
python -m pip install --upgrade pip
pip install -r requirements.txt
```

**Проблема:** Порт 5000 занят

```
Решение: Измените порт в app.py
# Найдите в конце файла:
app.run(debug=True, host='127.0.0.1', port=5000)
# Измените на:
app.run(debug=True, host='127.0.0.1', port=8000)
```
