## 🌐 REST API Endpoints

### Caesar Cipher

```
POST /api/caesar/encrypt
Body: { "text": "Hello", "shift": 3 }
Response: { "encrypted": "Khoor", "steps": [...] }

POST /api/caesar/decrypt
Body: { "text": "Khoor", "shift": 3 }
Response: { "encrypted": "Hello", "steps": [...] }
```

### AES Cipher

```
POST /api/aes/encrypt
Body: { "text": "Secret", "password": "mypassword" }
Response: { "encrypted": "gAAAA...", "steps": [...] }

POST /api/aes/decrypt
Body: { "encrypted": "gAAAA...", "password": "mypassword" }
Response: { "decrypted": "Secret", "steps": [...] }
```

### RSA Cipher

```
POST /api/rsa/generate-keys
Body: { "key_size": 2048 }
Response: { "public_key_generated": true, "steps": [...] }

POST /api/rsa/encrypt
Body: { "text": "Secret" }
Response: { "encrypted": "ac8f...", "steps": [...] }
```

### Algorithms Info

```
GET /api/algorithms
Response: {
  "caesar": { "name": "...", "description": "..." },
  "aes": { "name": "...", "description": "..." },
  "rsa": { "name": "...", "description": "..." }
}
```

## 🚀 Производительность

### Время выполнения

- **Caesar** — < 1 ms (для текста в 100 символов)
- **AES** — 10-50 ms (зависит от длины текста)
- **RSA** — 100-500 ms (генерация ключей 2048 бит может занять 5-10 сек)

### Использование памяти

- **Flask сервер** — ~30 MB
- **Криптографическая библиотека** — ~5 MB
- **Общее использование** — ~40-50 MB
