## 🚀 Как использовать

### Способ 1: Самый быстрый (Windows)

```bash
run.bat
```

### Способ 2: Ручной запуск (все ОС)

```bash
python -m venv venv
source venv/bin/activate  # или: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Затем откройте браузер

```
http://127.0.0.1:5000
```

### ✅ Что использовано

- Официальная криптографическая библиотека Python
- Проверенные фреймворки (Flask)

### ⚠️ Что помнить

- Это приложение для **обучения**
- Не используйте Caesar для реального шифрования
- Для production используйте HTTPS
