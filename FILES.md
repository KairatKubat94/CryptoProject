# 📦 Содержание проекта

## Созданные файлы

### 🔧 Backend файлы

#### [app.py](app.py) — Flask приложение

- **Размер:** ~4 KB
- **Строк:** 140
- **Описание:** Основной бэкенд сервер
- **Функции:**
  - Запуск Flask сервера на http://127.0.0.1:5000
  - REST API endpoints для всех алгоритмов
  - CORS поддержка для фронтенда
  - Обработка HTTP запросов (POST/GET)
  - Возврат JSON ответов

#### [crypto_algorithms.py](crypto_algorithms.py) — Реализация алгоритмов

- **Размер:** ~12 KB
- **Строк:** 380
- **Описание:** Криптографические алгоритмы
- **Классы:**
  - `CaesarCipher` — шифр подстановки
  - `AESCipher` — симметричное шифрование
  - `RSACipher` — асимметричное шифрование
- **Методы:**
  - `encrypt()` / `decrypt()` для каждого
  - `get_info()` — информация об алгоритме
  - Пошаговая визуализация процесса

## 🚀 Как начать

### Вариант 1: Самый быстрый (Windows)

```bash
run.bat
```

### Вариант 2: Ручной запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Вариант 3: Linux/Mac

```bash
bash run.sh
```

## 🌐 После запуска

1. Откройте браузер
2. Перейдите на http://127.0.0.1:5000
3. Выберите алгоритм
4. Введите данные
5. Нажимите кнопку
6. Смотрите пошаговое объяснение
